# Routes
### Base URL
	https://newsletterapi22.herokuapp.com/
	
### News
	use /news?page=1 to request news where you can receive per page
	
### Most Read
	use /mostread?type=&page=1 to request the most read news

	Types: null, semana, mes, sempre
